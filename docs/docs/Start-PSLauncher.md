# Start-PSLauncher

## SYNOPSIS

Reads the config file and launches the gui

## SYNTAX

### __AllParameterSets

```
Start-PSLauncher [-ConfigFilePath] <String> [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION

Reads the config file and launches the gui


## EXAMPLES

### Example 1: EXAMPLE 1

```
Start-PSLauncher -ConfigFilePath c:\temp\config.json
```
