Start-PSLauncherColorPicker

SYNOPSIS

Launches a gui form to test and change the color of PSLauncher.

SYNTAX

Start-PSLauncherColorPicker [-ConfigFilePath] <String> [-WhatIf] [-Confirm] [<CommonParameters>]

DESCRIPTION

Launches a gui form to test and change the color of PSLauncher.
